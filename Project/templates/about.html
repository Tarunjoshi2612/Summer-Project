SENTIMENT ANALYSER

What is Sentiment Analyser

Sentiment Analyser is an NLP based tool which is used to analyse the review of user by identifying that a given input is a positive sentiment or negative sentiment. Here we have used multinomial NaiveBayes to train the model which gives the accuracy of 85%.
Some insights of the project:
1. Firstly we cleaned the data by removing stopwords.
2. Then, we applied PorterStemmer for stemming the data.
3. We used TfidfVectorizer for Feature Engineering on text data.
4. Then, we used multinomial NaiveBayes for training the data.
5. Finally we checked the score of our data by cross_val_score.
